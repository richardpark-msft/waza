## Current Release: v0.9.0

**Released:** 2026-02-23

### What's New

#### Added

- **A/B baseline testing** — `--baseline` flag runs tasks with and without skill, computes improvement scores
- **Pairwise LLM judging** — `pairwise` mode on `prompt` grader with position-swap bias mitigation
- **Tool constraint grader** — `tool_constraint` type with `expect_tools`, `reject_tools`, `max_turns`, `max_tokens`
- **Auto skill discovery** — `--discover` flag walks directories for SKILL.md + eval.yaml

#### Fixed

- Resolved errcheck and ineffassign lint warnings

#### Changed

- Added competitive analyses (OpenAI Evals, skill-validator) and eval registry design doc
- Converted remaining ASCII diagrams to Mermaid

## Quick Install

<Tabs>
<TabItem label="macOS / Linux">

```bash
curl -fsSL https://raw.githubusercontent.com/spboyer/waza/main/install.sh | bash
```

</TabItem>
<TabItem label="Windows (PowerShell)">

```powershell
iwr -useb https://raw.githubusercontent.com/spboyer/waza/main/install.ps1 | iex
```

</TabItem>
</Tabs>

Verify the installation:

```bash
waza --version
```

## azd Extension

Install waza as an [Azure Developer CLI](https://learn.microsoft.com/azure/developer/azure-developer-cli/) extension.

**Current version:** v0.9.0

```bash
azd extension install azd-waza \
  --source https://github.com/spboyer/waza/releases/download/azd-ext-microsoft-azd-waza_0.8.0/azd-ext-microsoft-azd-waza_0.8.0.tar.gz
```

Once installed, run evaluations through `azd`:

```bash
azd waza run eval.yaml
```
